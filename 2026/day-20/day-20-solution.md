# Day 20 - Bash Log Analyzer and Report Generator


## Project Overview

This project analyzes server log files using Bash scripting.

The script:

- Validates input log file
- Counts errors
- Finds critical events
- Generates summary reports
- Archives processed logs


## Tools Used

### grep

Used for searching ERROR and CRITICAL messages.


### awk

Used for extracting error messages.


### sort

Used for sorting messages.


### uniq

Used for counting duplicate errors.


### wc

Used for counting total lines.
